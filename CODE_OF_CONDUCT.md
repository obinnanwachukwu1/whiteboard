# Code of Conduct

## Our Standards
We want Whiteboard to be a welcoming and professional community.

Examples of behavior that contributes to a positive environment:
- Being respectful and constructive
- Giving and receiving feedback in good faith
- Prioritizing user safety and privacy

Examples of unacceptable behavior:
- Harassment, personal attacks, or discriminatory language
- Doxxing, intimidation, or threats
- Deliberate disruption of discussions or contribution workflows

## Enforcement
Project maintainers are responsible for clarifying and enforcing this code of conduct.

To report unacceptable behavior, contact:
- [@obinnanwachukwu1](https://github.com/obinnanwachukwu1)

Maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.
