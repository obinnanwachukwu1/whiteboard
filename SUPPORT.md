# Support

## Getting Help
- Questions and usage help: open a GitHub Discussion (if enabled) or issue.
- Bug reports: use the Bug Report issue template.
- Feature ideas: use the Feature Request issue template.

## Scope
Whiteboard support focuses on:
- The latest release
- The current `main` branch

Older versions may receive limited support.

## Security Issues
For vulnerabilities, follow `SECURITY.md` and report privately.
