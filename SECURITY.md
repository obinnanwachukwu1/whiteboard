# Security Policy

## Supported Versions
Security fixes are applied to the latest development branch and latest release.

## Reporting a Vulnerability
Please do not report security issues in public issues.

Use one of these private channels:
1. GitHub Security Advisory (preferred)
2. Direct contact via GitHub: [@obinnanwachukwu1](https://github.com/obinnanwachukwu1)

Include:
- Affected version/commit
- Reproduction steps
- Impact assessment
- Suggested mitigation (if known)

We will acknowledge receipt and work on triage as quickly as possible.
